## Project Overview

This project is designed to monitor the level of water in a container using an Arduino UNO. Different LEDs are used to represent low, medium, and full water levels, while a buzzer provides an alert when the tank reaches the maximum level. The system helps in preventing water overflow and supports efficient water management.

## Working Principle

The Arduino continuously checks the input from the sensor/potentiometer. Based on the detected value:

* **Low Level:** Red LED glows
* **Medium Level:** Yellow LED glows
* **High/Full Level:** Green LED glows and buzzer activates

This provides a clear visual and sound indication of the water level inside the tank.
